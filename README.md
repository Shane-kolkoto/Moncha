# Moncha

## CRUD Stickers
    - Look in server folder to get database layout

## Install dependencies
    - npm i / npm install
    - yarn install

## Create .env 
    - look at .env.example file

## Run database
    - npx knex migrate:latest
    - npx knex seed:run

## Start program
    - npm run start ( run bin file )
    - npm run dev ( run nodemon)
    -npm run test ( test the database )
